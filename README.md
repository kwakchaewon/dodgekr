## 🎮 서비스 소개
리그오브레전드 커뮤니티 **Dodge.kr** 입니다.<br/>
Riot API 기반 전적 검색, 커뮤니티 등의 기능을 제공합니다.<br/>
<br/>
## 🛠  사용기술
### **Frontend**
- Tymeleaf

### **Backend**
- Java, Spring boot, Spring security
- 리그오브레전드 API ([링크])(https://developer.riotgames.com/))

### **Infra**
- Server: AWS EC2
- DB: AWS RDS (MYSQL)
